# 📊 Regional Sales Performance Analysis

---

## 📌 Project Overview

The **Regional Sales Performance Analysis** project is an end-to-end **Business Intelligence** solution developed as part of a **Data Analytics Internship at Skoliko**.

The project analyzes regional sales data of a US-based enterprise to identify sales trends, evaluate regional performance, measure profitability, and support data-driven business decisions.

Using **Python** for data preprocessing, **SQL** for data analysis, and **Microsoft Power BI** for interactive visualization, the project transforms raw sales data into meaningful insights through interactive dashboards and reports.

---

## 🎯 Objectives

- Analyze regional sales performance across different regions and states.
- Identify top-performing products and customer segments.
- Compare sales performance against budgets.
- Evaluate revenue and profit trends.
- Build interactive Power BI dashboards.
- Generate actionable insights to support business decision-making.

- Customers
- Products
- Regions
- State Regions
- Budget Data

### Key Fields

| Field | Description |
|--------|-------------|
| Order Number | Unique order ID |
| Order Date | Date of purchase |
| Customer Name | Customer details |
| Product Name | Product information |
| Sales | Total sales amount |
| Profit | Profit earned |
| Region | Sales region |
| State | State information |
| Order Quantity | Quantity sold |
| Budget | Budget target |
