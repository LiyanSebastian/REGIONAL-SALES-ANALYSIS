#📊 Regional Sales Performance Analysis

📌 Project Overview

The Regional Sales Performance Analysis project is an end-to-end Business Intelligence solution developed as part of a Data Analytics Internship at Skoliko. The project analyzes regional sales data of a US-based enterprise to identify sales trends, evaluate regional performance, measure profitability, and support data-driven business decisions.

Using Python for data preprocessing, SQL for data analysis, and Microsoft Power BI for interactive visualization, the project transforms raw sales data into meaningful insights through dashboards and reports.
